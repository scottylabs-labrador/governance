
# Synchronizer

This directory contains the Python-based permission synchronizer for ScottyLabs.

## Slack

- Create a Slack channel for each team and invite team members to the channel.

## Infra

- Automatically update the [`infra/inputs.json`](../../infra/inputs.json) file.

## CODEOWNERS

- Automatically update the [`CODEOWNERS`](../../.github/CODEOWNERS) file.
