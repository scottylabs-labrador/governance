# Joining as a contributor

> [!NOTE]
> This is for individuals seeking to register themselves as a contributor.

Create a new TOML file in `contributors/` with your GitHub username as the filename:

```toml
full-name = "Your Name"
andrew-id = "your-andrew-id"
```

Visit the [contributor schema](../../meta/schemas/contributor.schema.json)
to learn more about each field.
